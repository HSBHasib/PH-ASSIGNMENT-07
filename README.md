## Project Name: KeenKeeper
```js
Description: 
KeenKeeper is a simple web app it helps users manage and track their friendships.
Users can see friend details, add interactions like calls or text,
and view their communication history easily.

Technology what i use: 
React.js
React Router DOM
Tailwind CSS, DaisyUI
JavaScript (ES6+)
React-Toastify (NPM Package)
JSON (for product data)
React icons
Recharts

Features about my project.
1. Track interactions (Call, Text, Video) with friends
2. View timeline history of all activities
3. See friendship analytics with charts

```
